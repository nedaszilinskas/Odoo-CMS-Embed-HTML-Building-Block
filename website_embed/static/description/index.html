SaaS Deployment Instructions

1. To deploy on Odoo SaaS first you would need to add your user to "Technical Features" group.

2. Secondly you need to install "Base import module" (technical name: base_import_module) module which adds capability to install external modules.

3. At this point you should now be able to import external modules. Click on "Import Module" link in administration menu. You should see a dialog asking to select a module for import. Select downloaded module archive file (.zip) and click "Import Module" button. Thats it, enjoy!
